# Weekly Revenue Review Template  -  Dorje Teas

> Founder-facing weekly operating memo.
> To be completed every Monday morning before the week begins.
> All sections should be filled with real data once internal access is available.
> Sample entries below are illustrative only  -  marked [SAMPLE].

---

## How to Use This Template

This is not a reporting document. It is a **decision document**.

Every section should end with either:
- A decision that was made
- A question that needs the founder's input
- An experiment that will be run this week

If a section ends with only numbers and no action, the review has not done its job.

## Review Checklist

Before this memo is shared with the founder, confirm:

- [ ] All revenue numbers are reconciled against Shopify
- [ ] All ad spend numbers are reconciled against platform dashboards
- [ ] Contribution margin calculation includes all variable costs, not just product COGS
- [ ] Every section ends with a decision, flag, or open question
- [ ] No metric is reported without a comparison (vs. last week or vs. target)
- [ ] Top 3 issues are specific and actionable, not vague observations
- [ ] Founder decisions are clearly separated from operational decisions
- [ ] Experiments have clear owners, metrics, and end dates
